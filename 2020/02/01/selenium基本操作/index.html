🎈selenium的介绍和相关操作

🎏启动浏览器

from selenium import webdriver
browser = webdriver.Firefox
# 此时将打开Firefox浏览器是一个新标签页。
browser.get('https://www.baidu.com')
# 新标签页变成百度首页

🎏寻找元素并进行相关操作

  • 寻找元素find_element_*与find_elements_*；前者只返回一个WebElement对象，代表页面中匹配查询的第一个元素；后者则是返回一个列表，包含页面所有匹配元素
方法名 返回的WebElement对象/列表
📌browser.find_element_by_class_name(name)
browser.find_elements_by_class_name(name) 使用CSS类name的元素
📌browser.find_element_by_class_selector(selector)
browser.find_elements_by_class_selector(selector) 匹配CSSselector的元素
📌browser.find_element_by_id(id)
browser.find_elements_by_id(id) 匹配id属性值的元素
📌browser.find_element_by_link_text(text)
browser.find_elements_by_link_text(text) 完全匹配提供的text的<a>元素
📌browser.find_element_by_partial_link_text(text)
browser.find_elements_by_partial_link_text(text) 包含提供的text的<a>元素
📌browser.find_element_by_name(name)
browser.find_elements_by_ame(name) 匹配name属性值的元素
📌browser.find_element_by_tag_name(name)
browser.find_elements_by_tag_name(name) 匹配标签name的元素
—
以上，除了*_by_tag_name()方法，其他所有方法的参数都是区分大小写的。如果页面上没有元素匹配该方法要查找的元素，会抛出异常NoSuchElement

🐱‍🏍重点补充——常见该异常的原因

① iframe/frame

  • 介绍
    在HTML中有三种结构特征：树结构、层次结构、框结构。
    iframe正是框结构中的一员。每个iframe中都是一个独立的沙箱，它们拥有自己的window以及DOM。
    框结构一共有三种分别是frame、iframe、frameset；其中frameset与普通标签无区别，进行正常定位即可；而iframe和frame则相当于在一个HTML中自己又起了一个HTML是需要切换的。
    示例可看https://developer.mozilla.org/zh-CN/docs/Web/HTML/Element/iframe

  • 切换 

    <html lang="en">
<head>
<title>FrameTest</title>
</head>
<body>
<iframe src="a.html" id="frame1" name="myframe"></iframe>
</body>
</html>

    🏐前进 

    bowser.switch_to.frame(reference)

    • 这个referen如果传入的是字符串则可以是iframe的id或者name也就是对应上文的myframe和frame1

    • 如果传入的是数字，则实际上就是index，索引。也就是对于整个HTML中的frame/iframe进行排序，切换到第几个iframe 突然想到一个问题，如果是一个iframe中还有一个iframe呢…所以这个应该是要切两次，这个索引也是平级之间

    • 如果这几个都不知道，还可以用过查找元素来找他 

      driver.switch_to.frame(driver.find_element_by_tag_name("iframe"))

      🏐后退

    • 切回主文档 

      driver.switch_to.default_content()

    • 切回到上一个文档

      driver.switch_to.parent_frame()

      目前我遇到的异常原因就是这一个，遇到其他再补充

🎏有了WebElement对象可以读取的属性或调用的方法示例

属性或方法 描述
tag_name 标签名，例如‘a’表示<a>
get_attribute(name) 该元素name属性的值
text 该元素内的文本，例如<span>hello</span>中的‘hello’
clear() 对于文本字段或文本区域元素，清除其中输入的文本
is_displayed() 如果该元素可见，返回True，否则返回False
is_enabled() 对于输入元素，如果该元素启用，返回True，否则False
is_selected() 对于复选框或单选框元素，如果被选中返回True，否则False
location 一个字典，包含键‘x’和‘y’，表示该元素在页面上的位置
—

🎏 提交表单、点击按钮

  • 🎐点击按钮click()
    对于WebElement对象有一个click()方法，模拟鼠标在该元素上点击。可用于链接跳转，选择单选按钮，点击提交按钮，或者除法该元素被鼠标点击时发生的任何事。
    eg.对于一段链接<a href='https://www.baidu.com' class = 'a'>
    当我们通过find_element_by_class_name('a')得到对象后，直接WebElement.click()

  • 🎐发送特殊键
    对于一些键盘击键比如键盘箭头键、回车和换行键、Home、End、PageUp/Down键。在selenium.webdriver.common.keys模块的属性中，由于名字长所以，我们可以通过import来引入from selenium.webdriver.common.keys import Keys
    光标当前不在文本字段中时，按下home/end键，将使浏览器滚动到顶部或底部。

    htmlele = browser.find_element_by_tag_name('html')
htmlele.send_keys(Keys.END/HOME)
    属性 含义
    Keys.DOWN/UP/LEFT/RIGHT 方向键
    Keys.ENTER/RETURN 回车和换行
    Keys.HOME/END/PAGE_UP/PAGE_DOWN home/end等键
    Keys.ESCAPE/BACK_SPACE/DELETE 见英文名可知
    Keys.F1/F2/… 见英文名
    Keys.TAB 见英文名

  • 🎐浏览器按钮 

    browser.back()
# 返回键
browser.forward()
# 前进键  
browser.refresh()
# 刷新  
browser.quite()
# 关闭窗口
