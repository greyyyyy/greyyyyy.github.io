利用webbrowser打开网页

简单介绍及思路分析

webbrowser.get(URL)

可以打开在命令行打开网页。
接下来我们利用一个小练习来体会一下
编写一个脚本使得能够在命令行实现搜索某个地区地图的目的
也就是，不通过浏览器搜索，而是在命令行中执行该程序，输入你想查找的地址，即可打开该处的地图。
再翻译一下就是要实现，你输入一个地址，利用 

webbrowser.get(URL)

得到在谷歌地图中搜索该地址的网页
从而我们只需要实现能够输入找到
又可以发现，虽然我们任意搜索一个地址时，URL非常的复杂，但是我们只输入它的地点，仍然是可以得到相同的结果的。

以 

870 Valencia St,San Francisco, CA 94110

为例。当我们直接在谷歌地图中搜索时，得到的是

https://www.google.com/maps/place/870+Valencia+St,+San+Francisco,+CA+94110%E7%BE%8E%E5%9B%BD/@37.7589579,-122.4238514,17z/data=!3m2!4b1!5s0x808f7e3dae6df76b:0x6240f16ee9572080!4m5!3m4!1s0x808f7e3dae0fc797:0x26acf7c8a5797e94!8m2!3d37.7589579!4d-122.4216627

但实际上我们直接输入

https://www.google.com/maps/place/870+Valencia+St,+San+Francisco,+CA+94110

也可以得到相同的结果。

所以最后一步的代码可以确定下来是

webbrowser.open('https://www.google.com/maps/place/'+address)

那么我们得到这个address是我们在命令行中输入或者*复制到了剪贴板
也就是要用到 

sys.argv

与 

pyperclip.paste()

接下来我们将简单介绍一下这两个的用法。

简单介绍sys.argv与pyperclip.patse()以及字符串一些处理

sys.argv

实质上就是外部参数传入程序内部然后输出。
在命令行中👇运行这个程序，后面跟着的就是传入程序的参数。

记录一个
在Git Bash中运行.py 应该是 python name.py
在Windows的命令行中直接在该文件所在目录打开 name.py即可

也就是在命令行中运行的时候传入参数，即是，参数其实不在代码中，而在于你准备运行时传入的。
👇👇👇 

python name.py 参数

将以列表形式储存，并以空格分开每一个，所以，可以索引。下面是例子解释

try.py 的代码如下 

import sys
a = sys.argv
b = sys.argv[0]
c = sys.argv[1]
d = sys.argv[2]
print(a,b,c，d)


如上图所示

  1. 返回了一个列表
  2. 列表的第一个是它本身
  3. 列表中的每一个元素是以空格隔开来得到的

pyperclip.paste()

这个实际上就是粘贴，放入一个变量即可 

import pyperclip
a = pyperclip.paste()

a 里面就装了刚刚复制的内容了。

一些字符串的处理记录
‘分隔物’.join()
join后的括号里面即是可以迭代的，比如列表，然后以前面引号内的进行链接，形成字符串。
str.split()
通过在括号内设定分隔符，将字符串进行分割，返回一个列表。

可以看到

  1. 只用空格，全部分隔，则不需要引号
  2. 后面可以制定分隔次数，则前面的分隔符指定应该是一定要打上引号

    add = pyperclip.paste()
    address = '+'.join(add.split( ))

requests 下载文件

requests 实际上由两个抓取，一个是Get，一个是POST，在这里先写Get,后面再继续补上。

requests.get(URL)

实质上得到的就是该网页的源代码。
当然，我们可以通过这种方法来下载某个网页里的一篇文章，即是获得HTML里某些标签的内容即可。
在这里，主要今天的重点在于如果有一个txt可以下载的网址。

import requests
res = requests.get(URL)
type(res)

这个返回的是Response对象，其中包含了Web服务器对请求做出的响应。

👆👆👆

  1. Response 对象有一个status_code 属性，可以检查她是否等于requests.codes.ok,了解下载是否成功。
  2. 检查成功的另一种方法就是在Response对象上调用raise_for_status()方法。如果下载文件出错，这将抛出异常。如果下载成功，就什么也不做。

接下来就是我得到后，将要保存。

file = open('name.txt','wb')
# 必须用"写二进制"模式打开该文件，即向函数传入字符串'wb',作为open()的第二参数。
# 即是该页面是纯文本，也需要写入二进制数据，而不是文本数据，目的是为了保存该文本的
# "Unicode编码"
for i in res.iter_content(number):
    file.write(i)

👆👆👆

  1. res.iter_content(number)
    是指在循环的每次迭代中，返回一段内容。每段内容都是bytes数据类型，你需要指定一段内容包含多少字节。也就是number处的内容
  2. 结果将返回写入字节数和最后所剩的字数，如图👇

    可以看到，最后剩余字数并不是指，没达到number所设定的就不能够写入，只是给你的提示，相当于，‘噢，你看，你没有整除成功，还有余呢’

创建hexo的过程

Posted on 2020-01-22

Hexo博客创建记录

第一步 准备

  1. 准备node.js

    安装完后会有两个组件一个是node.js本身，另一个是npm包管理器。
    可以来查看一下node 与 npm 的版本

  2. 安装cnpm
    利用npm装淘宝的源

  3. 利用cnpm安装hexo 

cnpm install -g hexo-cli

一样可以利用 

hexo -v

查看版本顺便验证是否成功

  1. 建立一个目录

    mkdir blog

    这个名字随意

  2. 初始化博客
    切换到这个目录之下，然后 

    hexo init

到这儿我们就搭建基本完成了

可以用 

hexo s

来启动，即可得到一个地址，在浏览器中打开即可看到。

然后利用 

hexo n " 你自己去的文章名字"

来创建博文
然后利用VS code打开，再用markdown语法来写即可。
可以看到我们的文章都在/_posts/目录下

再在blog目录下进行生成 

hexo g

即可
此时再启动 

hexo s

就可以看到自己的博文

第二步 部署到远端(github)

在这里，我是将其部署到GitHub上的
首先我们新建一个仓库(repository)

在repository name 处一定写的是 

用户名.github.io

emmm我也不知道为什么，是这样说的。

创建成功后

在命令行中，在blog的目录下安装一个git 部署的插件 

cnpm install --save hexo-deployer-git

成功后需要修改👇这个文件

_config.yml

在git bash 里可以直接利用
vim _config.yml
就可以打开，再在里面修改成👇

即可
在Windows中退出
先按ESC 再
:wq 退出

修改完成后，
部署到远端
hexo d
期间会要求登录GitHub账号
登陆成功后，即可
成功后，再将仓库名在浏览器中访问即可。

第三步 换主题

在GitHub中找到自己喜欢的主题，再克隆到本地，要注意的是，在克隆中可能与GitHub上写的不太一样。(至少我是/(ㄒoㄒ)/~~)

可以看到在clone那里的地址末尾是没有
.git的，应该加上。
在我之前没加的时候就一直报错，加了之后才成功了。

clone成功了，就继续打开
_config.yml文件，在里面，修改，theme
将theme改为你的主题的名字(以前默认landscape)
值得注意的是，修改完之后，还没有被部署到远端。所以，最后还要记得
hexo d
才会推到远端

以上就是整个基本操作完成啦:)

Markdown基本语法记录

Posted on 2019-12-16 Edited on 2020-01-29

一、标题

设置为标题的文字前面加 # 来表示，一定要注意的是#后是有空格的。

一级一个#最大

二级两个#稍小

三级四级以此类推，最多六个。

二、字体

  • **加粗**

    要加粗的文字左右两边分别用两个*包起来

  • *斜体*

    用一个*包起来

  • ***倾斜和加粗***

    用三个***包起来

  • ~~删除线~~

    每边用两个~~包起来

  • 下划线

<u>内容</u>

三、引用

在引用文字前加>即可；

也可以嵌套，即是引用的引用则多加一个

比如

你好

你好

并且还要注意的是引用要在开头

四、排版问题

(可能也是我没多查其他的博文，我找的博文并没有说明这一点。)

我发现，如果要实现提行写，要在后面两个以上空格加回车
空一行则会，额，空一行
并且，在分隔符的上面如果不分开，会自动变大加粗，效果有点像二级标题的样式。

然后分隔的用

👉 —

👉 —-

👉 ***

👉 ****

然后两个内容之间只能用一个空格。你就算打无数个空格，显示出来的间距也最多是一个空格。

五、代码

用反引号包含``

代码块有两个方法

  • 一个tab
    如果用tab要注意，必须与前面非代码块空一行才能使用成功
  • 三个```;代码块前后用三个反引号包含

一定注意！第二种不能用在开头，它的使用前面同行必须有内容，不然无效，多次教训
